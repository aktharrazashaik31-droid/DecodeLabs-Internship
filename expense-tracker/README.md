Expense Tracker

Description

A simple Python application that allows users to enter multiple expenses and calculates the total amount spent.

Features

- Add multiple expenses
- Automatically calculates total spending
- Easy-to-use command-line interface

How to Run

1. Install Python 3.
2. Save the program as "expense_tracker.py".
3. Open Terminal/Command Prompt.
4. Run:

python expense_tracker.py

5. Enter expense amounts.
6. Type "done" to view the final total.
