To-Do List

Description

A simple Python To-Do List application that allows users to add multiple tasks and view them at the end. This project demonstrates the use of Python lists, loops, and user input handling.

Features

- Add multiple tasks
- Store tasks using Python lists
- Display all tasks entered by the user
- Simple command-line interface

Technologies Used

- Python 3
- Lists
- Loops
- User Input

How to Run

1. Install Python 3 on your system.
2. Save the program as "main.py".
3. Open Terminal or Command Prompt.
4. Navigate to the project folder.
5. Run the command:

python main.py
